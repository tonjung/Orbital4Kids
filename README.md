installation steps:
1) install jre from https://www.java.com/download/manual.jsp
2) download Orbital4Kids2004.exe  for Windows
3) download Orbital4Kids2004.jar  for other OS

How to run Orbitals4Kid on MacOS:
-locate and open Orbital4Kids2004.jar with javaLauncher from finder
-You need permission to open Orbital4Kids2004.jar in System Settings

All the steps are in https://www.youtube.com/watch?v=2Zk88zYJ3TU

Warning Orbital4Kids2004 was tested with Windows Enironment.   
I found that on MacOS X, please don't interact directly with orbital using mouse arrow, 
instead click on the control icons within the software.
